## 考试

### 2015级版本的范围

网工的专业选修课(必须选的那种) ，闭卷考试，主要是考卷1，有课设；  

TCP 状态机 ；连接管理：三次握手，四次挥手;     

SOCKET API 的使用及与TCP连接管理之间的关系;     

I/O 复用：select ，epoll；     

SOCKET options：fcntl；ioctl；      

udp 协议 ，编程基础；      

域名解析，地址转换函数使用；     

I/O 模型：阻塞、非阻塞；同步、异步；    

锁，条件变量，信号量 的基本概念与使用;   

POSIX 信号处理，signal，sigaction 的使用;    

守护进程，僵尸进程;     

高级I/O ：14 章；  

非阻塞 I/O：16 章；     

多线程，多进程；
