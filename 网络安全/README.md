## 课程相关

### 评价  

注重加解密算法，数字签名，认证和安全协议 部分，数学偏多(抽象代数)，有加密算法实验和课程设计，需要花较多时间认真学习。  

### 考试  

闭卷，需要认真过一遍 PPT 和笔记，挂科率较高。
